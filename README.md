# ScreenTime

Projeto em Java que simula o tempo de uso de aplicativos.

## 📌 Funcionalidades
- Cadastro de uso de aplicativos
- Cálculo de tempo total
- Identificação do app mais utilizado
- Relatório de uso

## 🛠️ Tecnologias
- Java
- Programação Orientada a Objetos
- ArrayList

## 🎯 Objetivo
Praticar lógica de programação e estruturação de sistemas com base em um problema real.

## ▶️ Como executar
1. Clone o repositório
2. Abra no IntelliJ
3. Execute a classe Main
